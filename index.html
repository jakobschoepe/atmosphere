ATMOSPHERE is an open-source platform for the design and conduct of randomized controlled trials. The main features that the platform currently offers are listed below:

  • feature Generating Randomization Schemes for Parallel Group Trial Designs
  • feature Enrolling through an Interactive Web Response System
  • feature Breaking Codes [IWRS]
  • feature Restricting Accessibility [IWRS]
  • feature Logging of User Activities With Timestamps [IWRS]
  • feature Deploying the Platform Using Features of the Docker Technology
  • performance Service is Running Stable in the Cloud
Deployment

To deploy the ATMOSPHERE platform on your local or virtual machine, please proceed as follows:

Step 0
Install the Docker Engine on your machine.

Step 1
Pull the image from Docker Hub by typing docker pull jakobschoepe/atmosphere in your command-line interface.

Step 2
Start the container by typing docker run jakobschoepe/atmosphere in your command-line interface.

🚀 Congratulations, you are running your own ATMOSPHERE platform on your machine. You can access ATMOSPHERE via http://localhost.
